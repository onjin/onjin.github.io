Cheat sheets

Swego czasu pojawiły się 'tablice informatyczne', będące kilkustronicowym skrótem dokumentacji oraz indeksem pojęć / funkcji, między innymi dla PHP (i nie pamiętam czego jeszcze).

Podobne zestawy możemy pobrać za darmo (PNG, PDF) ze strony http://www.ilovejackdaniels.com/cheat-sheets/

W momencie gdy to piszę dostępne są zestawy dla:

  • colorów RGB zapisanych heksadecymalnie
  • PHP
  • CSS
  • mod_rewrite
  • MySQL
  • JavaScript
  • HTML entities
  • ASP / VBScript
  • Ruby on Rails
  • World of Warcraft

Mała rzecz, a cieszy.
