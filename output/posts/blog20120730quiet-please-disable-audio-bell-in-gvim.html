quiet please !! disable audio bell in gvim

Posted: 2012-07-30 21:27   |  More posts about vim

I don't want to hear it any more, so I've put this into .vimrc

" quiet pleeeeeease

if has('autocmd')
  autocmd GUIEnter * set visualbell t_vb=
endif
