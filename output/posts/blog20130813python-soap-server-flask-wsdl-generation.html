Install

$ pip install Flask-Enterprise

file: soap.py

 1 from time import ctime
 2 from flask import Flask
 3 from flaskext.enterprise import Enterprise
 4 
 5 app = Flask(__name__)
 6 
 7 enterprise = Enterprise(app)
 8 
 9 
10 class DemoService(enterprise.SOAPService):
11     __soap_server_address__ = '/soap'
12     __soap_target_namespace__ = 'ns'
13 
14     @enterprise.soap(_returns=enterprise._sp.String)
15     def get_time(self):
16         return ctime()
17 
18 if __name__ == '__main__':
19     app.run(host='127.0.0.1', port=5555)

Run service

$ python soap.py

file: client.py

1 from suds.client import Client
2 
3 c = Client('http://127.0.0.1:5555/soap?wsdl')
4 print c.service.get_time()

Run test

$ python client.py
